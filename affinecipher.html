HOW TO USE THIS APP


CLICK ON AFFINE ENCRYPTION AND DECRYPTION TAB


After clicking you will be proceeded to affine cipher app part then enter your plaintext and select values for keys ie. a and b then click encrypt to get your cipher text. After getting your cipher text just click and see on plaintext tab first you entered it shows you the same message as plaintext you given


WORKING OF AFFINE CIPHER

ENCRYPTION


START OFF

C=(a.p+b)mod26 this the formula for encryption in which affine calculates cipher text. a->key1 b->key2 p->alphabet values of plaintext for eg:a->0 b->1...

HEART PART

WRITE THE CONVERTED CIPHERTEXT IN UPPER CASE LETTERS...ie.DKFKDSJFKDFJKDSJ WRITE IN THIS FORMAT

DECRYPTION


START OFF

P=a^-1(c-b)mod26 this the formula for decryption in which affine calculates plain text. a^-1->key1 inverse using euceldian algo, b->key2 c->alphabet values of ciphertext for eg:a->0 b->1...

HEART PART

WRITE THE CONVERTED PLAINTEXT IN SMALL CASE LETTERS...ie.pxkjkdjkdj WRITE IN THIS FORMAT
